## 1. Giới thiệu
React là thư viện JavaScript do Meta phát triển, tập trung vào UI dựa trên component.

## 2. Ngôn ngữ & nền tảng
- Ngôn ngữ: JavaScript, TypeScript
- Môi trường: Browser, Node.js

## 3. Kiến trúc
- Component-based
- Virtual DOM
- One-way data flow

## 4. Use case phổ biến
- Single Page Application (SPA)
- Dashboard
- Web application lớn

## 5. Hệ sinh thái
- Routing: React Router
- State: Redux, Zustand
- Framework liên quan: Next.js, Remix

## 6. Ưu điểm
- Cộng đồng lớn
- Dễ mở rộng
- Hiệu năng tốt cho UI

## 7. Hạn chế
- Không phải framework full-stack
- Cần nhiều thư viện phụ trợ

## 8. Ví dụ code

```jsx
function App() {
  return <h1>Hello React</h1>;
}

export default App;
```

## Ngôn ngữ sử dụng

- JavaScript
- TypeScript
